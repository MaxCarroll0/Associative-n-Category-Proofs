# Proofs with Semi-strict Associative n-Categories via homotopy.io
5 sample proofs about monoidal categories, monoidal 2-categories, knot theory via constructing semi-strict associative n-categories. 

Proved using the [homotopy.io](beta.homotopy.io) proof assistant.

Descriptions of each proof available in the 'abstract' of the .hom files (open in homotopy.io).
